# 动态规划（Dynamic Programming）

## 概念

动态规划是一种将复杂问题分解成子问题并保存子问题结果以避免重复计算的算法思想。通常用于求最优解（最大/最小）、计数、可行性等问题。

## 适用场景

- 问题具有**最优子结构**：全局最优包含子问题的最优解。
- 问题具有**重叠子问题**：子问题被多次重复求解。
- 常见类型：线性 DP、背包、区间 DP、状态压缩 DP 等。

## 基本思路

1. **定义状态**：用 dp[i] 或 dp[i][j] 表示某种含义（如到第 i 步的最优值）。
2. **状态转移方程**：由已知状态推出新状态。
3. **边界与初始化**：确定初始状态（如 dp[0]、dp[1]）。
4. **计算顺序**：按依赖关系确定循环顺序，保证递推时依赖项已计算。

## 简单例子：斐波那契

- 状态：dp[i] = 第 i 个斐波那契数。
- 转移：dp[i] = dp[i-1] + dp[i-2]。
- 边界：dp[0]=0, dp[1]=1。

## 与贪心、分治的区别

- **贪心**：每步做当前最优选择，不回溯；不保证全局最优，除非具有贪心选择性质。
- **分治**：子问题独立、不重叠，通常不记录子问题解。
- **DP**：子问题重叠，用表记录子问题解，避免重复计算。

## 做题建议

从简单线性 DP、背包开始，再练区间 DP 和状态压缩；写清状态定义与转移，注意边界与下标。
