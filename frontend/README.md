# SynCode Frontend Workspace

This workspace contains the product-grade SynCode frontend shell used to preview the brand, layout, and workflow before real backend integration.

## Apps

- `@aioj/web`: SynCode landing site
- `@aioj/app`: user-facing SynCode app
- `@aioj/admin`: SynCode admin console

## Packages

- `@aioj/tokens`: shared design tokens
- `@aioj/ui`: shared UI primitives
- `@aioj/api`: mock contracts and query helpers
- `@aioj/config`: shared product content and navigation config

## Local Commands

```bash
npm install
npm run dev:web
npm run dev:app
npm run dev:admin
```

## Build Commands

```bash
npm run build:web
npm run build:app
npm run build:admin
```
